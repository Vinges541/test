# Подключение к БД
Необходимо скопировать файл example.env и дать ему имя .env, после чего указать в нем необходимые параметры для подключения
# Локально

## Установка
```bash
$ npm install
```

## Запуск
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

# или через Docker
```bash
$ docker-compose up -d
```

По умолчанию путь до Swagger UI - http://localhost:3000/api
